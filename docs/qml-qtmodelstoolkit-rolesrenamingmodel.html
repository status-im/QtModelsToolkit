RolesRenamingModel QML Type

Simple proxy allowing to rename roles according to a provided mapping. More...

Import Statement: import QtModelsToolkit .
  • List of all members, including inherited members

Properties

  • mapping : list<RoleName>

Detailed Description

In the following example original roles key and title are renamed to id and name, respectively.

RolesRenamingModel {
    id: renamedModel

    sourceModel: sourceModel

    mapping: [
        RoleRename {
            from: "key"
            to: "id"
        },
        RoleRename {
            from: "title"
            to: "name"
        }
    ]
}

Property Documentation

mapping : list<RoleName>

This property holds the mapping as a list of RoleRename objects.
